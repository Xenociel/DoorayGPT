해당 프로젝트는 파이참 환경에서 작성되었습니다.
파이썬은 3.9.13버전을 사용하였으며, FastApi 프레임워크를 이용하였습니다.


<파이썬 버전 확인>
- 터미널에서 Python -V를 입력하여 현재 파이썬의 버전을 확인해주세요.
  = 일반적인 리눅스 환경의 경우 기본 버전은 python 2.7.X로 되어 있으므로 
      해당 링크를 참조하여 파이썬을 3.9.13으로 업그레이드 해주세요.
    https://www.zinnunkebi.com/upgrade_python27to36/

<실행>
1. 압축을 해제하고 프로젝트 디렉토리를 열어주세요
2. 프로젝트 디렉토리에서 터미널을 열어 아래 명령어를 입력한다.
   pip install -r requirements.txt
3. 실행
   3-1. 소스 코드 기준 터미널을 이용하여 main.py가 위치한 폴더에 접근 후 해당 명령어를 이용하여 실행
         uvicorn main:app --host [설정한 IP] --reload --port=[설정한 포트]
   3-2. 또는 파이참 환경에서 main.py를 run 하여 실행.
        이 경우 .env 파일 내 IP와 PORT를 수정 후 실행
4. http://[설정한 IP]:[설정한 포트]/docs# 또는 http://[설정한 IP]:[설정한 포트]/redoc 에 접속하여 스웨거 문서 확인

<사용법>
1. 우측상단의 채널별 멤버/설정 버튼을 누르고 설정 탭으로 이동하여 서비스 연동을 클릭합니다.

2. (연동 중인 서비스에 'incoming'이 있는 경우 3번을 수행합니다.)서비스 추가 탭으로 이동하여 스크롤을 내린 후 'incoming' 서비스를 추가해 줍니다.

3. 연동 중인 서비스 탭으로 이동하여 우측의 '연동 URL'을 클릭하여 URL을 클립보드에 복사합니다.

4. 채팅 창으로 돌아가 입력창에 '/gptset [복사한 연동 URL]' 을 입력하고 엔터를 누릅니다.

5. 'Chat Answer URL successfully set' 메세지가 출력되었다면 세팅이 완료되었습니다.

6. GPT와 대화를 시작하려면 '/gpt [대화 내용]'으로 시작하십시오
